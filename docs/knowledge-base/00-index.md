# Knowledge Base Index

Failure modes currently documented under `failure-modes/`. This index is itself
retrievable; keep entries short.

## Solder-joint defects

- **Opens** (`failure-modes/opens.md`) — no electrical connection where one is expected.
- **Shorts / solder bridges** (`failure-modes/shorts.md`) — unintended connection between nets.
- **Cold solder joint** (`failure-modes/cold-solder-joint.md`) — dull, high-resistance joint.
- **Insufficient solder** (`failure-modes/insufficient-solder.md`) — too little fillet, marginal joint.

## Placement defects

- **Tombstoning** (`failure-modes/tombstoning.md`) — chip component lifted on one end.
- **Component misorientation** (`failure-modes/component-misorientation.md`) — wrong polarity / rotation.
- **Missing component** (`failure-modes/missing-component.md`) — part absent from the board.

## Parametric defects

- **Out-of-tolerance analog** (`failure-modes/out-of-tolerance-analog.md`) — measured value drifts outside limits.
